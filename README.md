# Shadow Kingdom: Bastion

A portrait mobile-first Tower Defence prototype built with Phaser 3, TypeScript, Vite, and Vitest.

## Play online

The playable demo is deployed with GitHub Pages:

https://xupiter01.github.io/shadow-bastion/

## Local development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run build
```

## Scope

Milestone 1 includes one map, five waves, fixed tower placement, three tower types, enemy waves, win/lose/restart flow, and debug controls. It is a prototype; mobile touch and FPS still require device-level QA.
